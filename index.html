TextUML

© 2011 Yuri Khan
http://github.com/yurivkhan/textuml

A collection of scripts to produce UML diagrams from a text description using Graphviz.

Documentation

  • Rationale
  • deps2dot, dependency graphs
  • classes2dot, class diagrams
  • states2dot, state diagrams

Download

This project is currently an early alpha, so no downloadable packages have been released. You can clone the project with Git by running:

$ git clone git://github.com/yurivkhan/textuml
